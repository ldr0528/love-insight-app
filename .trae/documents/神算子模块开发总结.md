# 神算子 (Divine Oracle) 模块上线计划

## 功能概述
我们已成功实现了“神算子”模块，旨在为用户提供针对具体问题的“一事一测”服务。该模块采用了独特的“赛博玄学”设计风格，结合了传统签文与 AI 智能解读。

## 核心功能点
1.  **Home 页入口**：
    *   新增深色主题的“神算子”卡片，带有动态旋转的罗盘图标，营造神秘感。
    *   入口文案：“心中若有惑，天机自会应”。
2.  **神算子页面 (`DivinePage`)**：
    *   **沉浸式 UI**：深空背景 + 金色符文 + 动态光效，打造“连接宇宙能量”的仪式感。
    *   **三段式交互**：
        1.  **心诚则灵**：用户输入问题（或选择热门问题），按下“发送”键。
        2.  **天机推演**：3秒钟的罗盘旋转与能量汇聚动画，模拟“起卦”过程。
        3.  **指引显现**：展示卦签（如“第十五签 · 上上签”）、签诗以及 AI 对该问题的深度解读。
3.  **后端逻辑**：
    *   新增 `divine_oracle` 报告类型。
    *   专门定制了 System Prompt，让 AI 扮演精通周易与塔罗的大师，针对用户具体问题进行解答。
    *   包含数据校验与 Mock 回退机制，确保服务稳定性。

## 技术实现细节
*   **前端**：使用 React + Tailwind CSS 实现复杂动画（`animate-spin`, `animate-pulse` 等）。
*   **后端**：复用 `POST /api/report` 接口，扩展 Schema 支持签文结构。
*   **路由**：新增 `/divine` 路由。
